# Символы псевдографики DOS (CP437)

## Обзор

Таблица символов кодовой страницы **CP437** (DOS / OEM), коды **176–220** —
диапазон заливок и линий рамок. Это ровно те символы, которыми нарисованы
диаграммы в [`scheme.md`](../docs/scheme.md).

- **Dec** — десятичное значение байта в CP437.
- **Hex** — то же в шестнадцатеричном виде.
- **Unicode** — кодовая точка, которую нужно сохранять в UTF-8-файле.
- 176–178 — блоки заливки; 179–218 — рамки (одинарные / двойные линии);
  219–220 — начало диапазона сплошных блоков.
